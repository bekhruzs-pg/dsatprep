# Non-Linear Functions

## 1. Overview
This broad topic covers Quadratic functions (parabolas), Exponential functions (rapid growth/decay), and general function notation. The DSAT loves to test your understanding of how equation parameters affect the graph (vertex, y-intercept, growth rate).

## 2. Key Concepts & Formulas
*   **Quadratic Forms**:
    *   **Vertex Form**: $y = a(x-h)^2 + k$. Vertex is $(h, k)$.
    *   **Intercept Form**: $y = a(x-p)(x-q)$. x-intercepts are $p$ and $q$.
    *   **Standard Form**: $y = ax^2 + bx + c$. Vertex x-coordinate is $-b/2a$.
*   **Exponentials**: $y = a(b)^x$
    *   $a$ = Initial value (y-intercept).
    *   $b$ = Growth factor ($1 + r$) or Decay factor ($1 - r$).
*   **Function Notation**:
    *   $f(x) + k$: Shift up $k$.
    *   $f(x - h)$: Shift right $h$.
    *   $f(g(x))$: Plug $g(x)$ into $f(x)$.

## 3. Step-by-Step Strategy
1.  **Identify the Function Family**: Is it $x^2$ (Quadratic) or $2^x$ (Exponential)?
2.  **Find Key Features**:
    *   **Quadratics**: Look for the vertex (max/min) or intercepts.
    *   **Exponentials**: Look for the initial value ($x=0$) and the ratio between consecutive terms.
3.  **Use Desmos**:
    *   Graph the function to see the vertex, intercepts, or transformations immediately.
    *   For function notation like "If $f(x) = ...$, what is $f(3)$?", just type the function and then type $f(3)$.

## 4. Common Pitfalls
*   **Vertex Sign**: In $y = (x-3)^2 + 5$, the vertex is $(3, 5)$, NOT $(-3, 5)$.
*   **Growth Rate**: If a population grows by 5%, the factor is $1.05$, NOT $0.05$.
*   **Linear vs. Exponential**: Linear adds a constant amount; Exponential multiplies by a constant factor.

## 5. Example Walkthrough
**Problem**: The function $f(x) = 2(x - 4)^2 + 3$ models a parabola. What is the minimum value of the function?

**Solution**:
1.  **Identify Form**: This is **Vertex Form** $a(x-h)^2 + k$.
2.  **Identify Vertex**: $(h, k) = (4, 3)$.
3.  **Identify Direction**: Since $a = 2$ (positive), the parabola opens **UP**.
4.  **Min/Max**: An upward-opening parabola has a **minimum** at the vertex.
5.  **Answer**: The minimum value is the y-coordinate, which is **3**.
